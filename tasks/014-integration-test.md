# Task 014: 핵심 기능 통합 테스트

## 상태: 완료 ✅

## 명세

전체 사용자 플로우, 관리자 플로우, 익명성 보장, 에러 핸들링, 다중 사용자 시나리오를 E2E 테스트한다.

## 영향 범위

- **QA**: E2E 테스트 (코드 변경 없음)

## 관련 기능

- F001~F011 통합 테스트

## 테스트 시나리오

1. 일반 사용자 전체 플로우 (가입 → 로그인 → 작성 → 조회 → 수정 → 삭제 → 로그아웃)
2. 관리자 플로우 (로그인 → 작성자 이름 확인 → CRUD)
3. 익명성 보장 (일반 사용자: Badge 미표시, 관리자: Badge 표시)
4. 에러 핸들링 (존재하지 않는 피드백, 타인 작성건 접근)
5. 탭 전환 + 검색 복합
6. 다중 사용자 시나리오 (A 작성 → B 수정 불가)

## 검증 방법

- 미들웨어 리다이렉트: curl HTTP 요청으로 자동 검증
- RLS 정책: Supabase SQL로 정책 등록 및 트리거 동작 검증
- 브라우저 플로우: 수동 브라우저 테스트 (Playwright MCP 미로드로 대체)
